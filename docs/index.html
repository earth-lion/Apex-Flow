المميزات

كل ما تحتاجه لإدارة أعمالك

نظام ERP متكامل يجمع بين الأمان المتقدم والذكاء الاصطناعي وسهولة الاستخدام

🔐
Role-Based Access Control
نظام صلاحيات متقدم بثلاثة مستويات (Admin / Manager / Cashier) مبني على Spatie Laravel Permission مع فلترة ديناميكية للقائمة الجانبية.
🤖
AI Business Copilot
مساعد أعمال ذكي مدمج في كل صفحة يعمل بـ Gemini API مع محرك احتياطي قاعدي للعمل بدون إنترنت، يُجيب عن المبيعات والمخزون.
⏰
Cashier Shift Management
نظام ورديات كاشير متكامل — فتح/إغلاق الورديات، تتبع الرصيد النقدي تلقائياً، حساب الفروق عند الإغلاق، وربط كل فاتورة بالوردية.
📊
Financial Reports & Analytics
تقارير مالية تفاعلية بالمخططات — إيرادات وتكاليف وأرباح مع مقارنة زمنية، مدعومة بـ Recharts ومحسّنة بـ Cache لأسرع أداء.
🧾
Invoice Engine + PDF
محرك فواتير يدعم البيع والشراء والمرتجع والهالك — مع توليد PDF احترافي بـ DomPDF وإرسال بريد تلقائي عبر قائمة انتظار.
📜
Visual Audit Trail
سجل تدقيق مرئي بتصميم Timeline — يعرض كل تعديل على قاعدة البيانات مع Diff Viewer يُقارن القيم القديمة والجديدة ملوّناً.
